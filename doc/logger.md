## Logger

Useful functions to log whatever you need.
Note that you shouldn't really use prints.

* `log(message)`
Adds `[+]` as prefix.
    * `message`: string
* `warn(message)`
Adds `[!] Warning: ` as prefix.
    * `message`: string
* `error(message)`
Adds `[-] Error: ` as prefix.
    * `message`: string
* `indent(message)`
Adds a tab as prefix.
    * `message`: string
